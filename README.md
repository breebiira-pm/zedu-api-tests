# 🧪 JSONPlaceholder API Test Automation

Automated API test suite for the [JSONPlaceholder](https://jsonplaceholder.typicode.com/) platform, built with **Python** and **Pytest**.

---

## 🌍 Project Overview
This project contains **30 automated tests** for the JSONPlaceholder REST API.  
It demonstrates how to validate API endpoints across multiple dimensions:

- ✅ Happy path responses (basic functionality works as expected)  
- ❌ Error handling (invalid requests return proper codes)  
- 🏠 Nested data validation (checking fields inside complex JSON objects)  
- 🔗 Chained requests (relationships between endpoints are consistent)  
- 🔍 Specific value checks (IDs, titles, emails, booleans are correct)  
- 🎁 Bonus tests (pagination, filtering, ownership checks, data type consistency)

This suite is designed to be **understandable even for non‑technical reviewers**: each test checks that the API behaves in a predictable, reliable way.
